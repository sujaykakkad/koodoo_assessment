### Installation

The application requires [Node.js](https://nodejs.org/) v12+ to run.

Install the dependencies.

```sh
$ cd koodoo_assessent
$ npm i
```
In order to test the function there are unit test cases which can be executed with this script.
```sh
$ npm run test
```
